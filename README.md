# HospitalManagementSystem

This repository contains my SQL Server coursework for a hospital management system.

At the current stage, the project focuses on the database design part of the system. The SQL script creates the main database schema for managing patients, doctors, appointments, medical services, payments, and prescriptions.

## Current Content

- `hospital_database.sql`  
  Creates the hospital database schema in Microsoft SQL Server.

## Database Tables

The current schema includes:

- Patients
- Doctors
- Appointments
- Medical_Services
- Payments
- Prescriptions

## Purpose

This project is being developed step by step as part of weekly lab work.  
Future weeks may include additional SQL operations and desktop application development connected to this database.

- Microsoft SQL Server
- SQL Server Management Studio (SSMS)
